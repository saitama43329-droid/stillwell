## Quick Start

Simply run the batch file:

```bash
run.bat
```

This will:
1. Install all dependencies (if needed)
2. Start the development server
3. Automatically open the website in your browser at http://localhost:3000

## Manual Commands

If you prefer to run commands manually:

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling

## Design Philosophy

StillWell embodies a premium, calm learning experience with:
- Serif typography for a classic, trustworthy feel
- Muted, earthy color palette for reduced eye strain
- Generous whitespace for clarity
- Smooth animations for engagement without distraction
